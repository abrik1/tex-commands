# tex-commands
Some useful commands for Texlive

# Mathematical Symbols:
- `$a^b$` is $a^b$ (exponents)
- `\int` is $\int$ (`$\int$`)
- `\circ` is used to make a circle or degree symbol such as  $180^{\circ}$ (`$180^{\circ}$`)
